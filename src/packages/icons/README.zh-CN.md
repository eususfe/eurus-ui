# EIcon 图标

## 引入

```javascript
import { EIcon } from 'eurus-ui';

Vue.createApp().use(EIcon.name, EIcon)
```

### EIcon Props

| 名称 | 类型 | 默认值 | 说明 |
| --- | --- | --- | --- |
| name | string | - | icon name |
| color | string | - | icon color |
| size | `'xs' \|'sm' \|'md' \|'lg' \|'xl'`｜number |  | icon size |
